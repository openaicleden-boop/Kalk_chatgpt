# Инженерный калькулятор

Небольшой веб-сервис с инженерным калькулятором: поддерживаются степени, тригонометрия,
логарифмы, константы π и e, а также история вычислений.

## Запуск локально

```bash
python3 -m http.server 8000
```

Откройте в браузере `http://localhost:8000`.

## Деплой на GitHub Pages

1. В репозитории откройте **Settings → Pages**.
2. В разделе **Build and deployment** выберите **Deploy from a branch**.
3. Укажите ветку (обычно `main`) и папку `/ (root)`, затем сохраните настройки.
4. После завершения сборки страница будет доступна по адресу вида `https://<username>.github.io/<repo>/`.

Так как проект статический (HTML/CSS/JS лежат в корне), дополнительных сборочных шагов не требуется.
